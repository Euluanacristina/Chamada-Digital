# Chamada Digital

Sistema de chamada online para professores universitários.

## O que é

Chamada Digital é um aplicativo web para professores registrarem a presença dos alunos de forma rápida e organizada. Funciona como PWA — pode ser instalado no celular ou computador e funciona sem internet.

## O que dá pra fazer

- Cadastrar múltiplas turmas e alunos
- Realizar chamada marcando Presente, Falta ou Justificada
- Adicionar justificativa para faltas
- Consultar e editar chamadas anteriores
- Gerar relatórios de presença exportados em PDF ou Excel
- Imprimir relatórios diretamente pelo app

## Como funciona o cálculo de faltas

Cada falta vale 5%. O limite padrão é 25%, ou seja, o aluno entra em situação crítica ao atingir 5 faltas. O limite pode ser ajustado por turma.

## Dados

Todos os dados ficam salvos localmente no dispositivo. Nenhuma informação é enviada para servidores externos.

## Como rodar

```bash
npm install
npm run dev
```
